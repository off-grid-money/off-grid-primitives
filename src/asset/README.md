## Submodules

- **Spot** — `asset::spot`
- **Futures** — `asset::futures`
- **Option** — `asset::option`

## Overview

This module provides asset primitives used across Off-Grid (e.g. asset ids, metadata). It is structured by product type: spot, futures, and options.

// When exposed via lib.rs:
